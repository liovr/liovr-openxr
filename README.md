# liovr-openxr
<https://github.com/liovr/liovr-openxr>

莉鸥的 [OpenXR](https://www.khronos.org/OpenXR/) 实现

[![CI](https://github.com/liovr/liovr-openxr/actions/workflows/ci.yml/badge.svg)](https://github.com/liovr/liovr-openxr/actions)


## 编译开发

+ 代码格式化:

  ```
  cargo fmt
  ```

+ 编译:

  ```
  cargo build
  ```

+ 测试:

  ```
  cargo test
  ```

+ 代码文档:

  ```
  cargo doc
  ```
